# 変更履歴

memologの主要な機能追加・改善をバージョンごとにまとめています。

## v0.0.13 (タイムゾーン対応と品質向上)
- **タイムゾーン処理の改善**: ISO 8601形式(タイムゾーンオフセット付き)でタイムスタンプを保存
  - ユーザーのローカルタイムゾーンで投稿時刻を正確に記録
  - 異なるタイムゾーンのユーザー間でも投稿時刻を正確に共有
  - 夏時間(DST)の切り替えに対応
  - UTC形式との後方互換性を維持
- **CI環境互換性の確保**: ローカル(GMT+9)とCI(GMT+0)の両環境で一貫した動作を実現
  - 環境依存テストの完全修正
  - タイムゾーン非依存のテンプレートパース処理実装
- **コード品質の大幅向上**: Lintエラー42個を完全解決
  - 型安全性の向上(non-null assertion除去、ジェネリック型指定)
  - コードの不変性保証(let → const)
  - 意図の明確化(不要なasync削除)
- **テスト品質の向上**: 全865テスト成功率100%達成
  - Node.js 18.x & 20.x 両環境で成功 ✅
  - テストカバレッジ95.19%維持
